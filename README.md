# 🩺 Breast Cancer Classification - ML Mid Project

## Overview
Binary classification project to predict whether a tumor 
is Malignant or Benign using 4 ML models.

## Dataset
Wisconsin Breast Cancer Dataset
🔗 https://archive.ics.uci.edu/ml/datasets/breast+cancer+wisconsin+(diagnostic)

## Models Used
- Decision Tree  → 91.23%
- Naive Bayes    → 93.86%
- KNN            → 95.61%
- ANN            → 96.49% 🏆 Best Model

## Files
- breast_cancer_classification.ipynb → Complete Code
- classification_report.pdf → Project Report

## Results
| Model         | Accuracy | F1-Score |
|---------------|----------|----------|
| Decision Tree | 91.23%   | 0.9118   |
| Naive Bayes   | 93.86%   | 0.9383   |
| KNN           | 95.61%   | 0.9558   |
| ANN           | 96.49%   | 0.9647   |
